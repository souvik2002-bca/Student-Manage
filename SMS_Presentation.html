01 — Project Overview

What is this system?

Project Overview

The Student Management System (SMS) is a full-stack web application built on the MERN stack that digitizes all core academic operations for an educational institution.

  • Centralized management of students, teachers, courses & subjects
  • Role-based dashboards for Admin, Teacher, and Student
  • Attendance tracking with daily mark/view workflows
  • Marks upload, grading & result reports per subject
  • Fee collection tracking with payment status management
  • Student ID Card generation with QR code (printable)
  • JWT-secured REST API backend with bcrypt password hashing

Technology Stack

MongoDB

NoSQL database — stores all data in JSON-like documents. 8 collections.

Express.js

Node.js web framework — handles REST API routing & middleware chain.

React 18

Frontend SPA — hooks, context API, React Router v6, Vite dev server.

Node.js

JavaScript runtime — runs the backend server on port 5000.

JWT Auth

JSON Web Tokens — stateless authentication with 7-day expiry.

bcryptjs

Password hashing — all user passwords stored as bcrypt hashes (salt 12).

Overview of assigned subjects, total students, today's pending attendance tasks and quick actions.

Authentication Flow

1

User submits username & password on Login Page

2

Frontend POSTs to /api/auth/login via Axios

3

Backend finds user by username, verifies bcrypt hash

4

JWT token signed with JWT_SECRET (7-day expiry)

5

Token stored in localStorage on client side

6

Axios interceptor injects Bearer token on every request

7

protect middleware verifies token on protected routes

8

authorize middleware checks role for restricted endpoints

Local Deployment Steps

PrerequisitesNode.js 18+, MongoDB running locally on port 27017
Install Backendcd backend && npm install
Configure .envMONGO_URI, JWT_SECRET, PORT=5000 — already set in backend/.env
Seed Databasenode resetAdmin.js
Start Backendnpm start → http://localhost:5000
Install Frontendcd frontend && npm install
Start Frontendnpm run dev → http://localhost:5173
